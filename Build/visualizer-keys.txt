Visualizer Help Menu (F1) - Page 1

ALT + ENTER             Toggle fullscreen (or double-click)
ALT + S                 Toggle multiple monitor stretch
SHIFT + UP / DOWN       Increase/decrease window opacity (or mousewheel)
                        
F1                      Show/hide help
F2 / 2xR MOUSEBUTTON    Toggle borderless window (unless in fullscreen/stretch mode)
CTRL + F2               Reset window
CTRL + SHIFT + F2       Set window to fixed dimensions from config file
F3                      Set fps to 30/60/90/120/144/240/360/Unlimited
CTRL + F3               Display current fps setting
F4                      Show/hide preset info
F5                      Show/hide fps
F6                      Show/hide rating
F7                      Always on top
F8 / R+M MOUSEBUTTON    Open Milkwave Remote
F9                      Toggle clickthrough mode
CTRL + F9               Toggle windowed fullscreen mode (cover screen except taskbar)
CTRL + SHIFT + F9       Toggle watermark mode (windowed clickthrough fullscreen)
F10 / CTRL + Z          Enable/disable Spout output / Holding SHIFT: set fixed Spout resolution
F11                     Hardcut mode: normal/based of music
F12                     Transparency mode (removes the black background)
CTRL + F12              Toggle black mode (no preset rendering)
                        
LEFT ARROW / Z          previous track
RIGHT ARROW / V         next track
DOWN ARROW / X          play/pause track
UP ARROW / C            stop track
CTRL + L/R ARROW        fast forward/rewind track (5 seconds)
B / M MOUSEBUTTON       show track info

CTRL + A                toggle next preset when track changes
CTRL + B                toggle track info polling
CTRL + C                toggle cover display when track changes

Visualizer Help Menu (F1) - Page 2

SPACE / H               Soft / hard cut to next preset
BACKSPACE               Go back to previous preset
` / ~                   [un]lock current preset
R                       Toggle random/sequential preset order 
L                       Load specific preset
M                       Show/hide preset-editing menu
N                       Show per-frame monitor
S                       Save new preset
CTRL + S                Quicksave preset to /presets/Quicksave folder
SHIFT + CTRL + S        Quicksave preset to /presets/Quicksave2 folder
CTRL + D                Set current audio device to default
SHIFT + CTRL + D        Display current audio device
CTRL + M                Toggle interative mouse control
CTRL + Q                Double quality factor (range 0.01 - 1.0)
SHIFT + CTRL + Q        Half quality factor (pixellize)
CTRL + H                Shift Hue

K                       Toggle sprite/message mode
SHIFT + K               Enter sprite kill mode
00-99                   Invoke sprites/messages
*                       Clear digits and reload messages.ini
DEL                     Clear latest sprite from the slot/message
SHIFT + DEL             Clear oldest sprite from the slot/message
SHIFT + CTRL + DEL      Clear all the sprites/messages
CTRL + K                Kills all the sprites
CTRL + T                Kills custom messages/song titles

W / w    Wave mode +/-              { / }    Y +/-
E / e    Wave alpha +/-             < / >    Rotation +/-
U / u    Wave warp scale +/-        A        Do random mini-mash-up   
I / i    Wave zoom +/-              D        Toggle shader locks   
O / o    Wave warp amount +/-       F        Tooggle echo orientation
P / p    Video echo alpha +/-       !        Randomize warp shader
G / g    Gamma adjustment +/-       @        Randomize comp shader   
J / j    Wave scale +/-
